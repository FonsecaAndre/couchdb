Processo de Instalação do CouchDB

Vídeo-apresentação Sobre

Instalação Tradicional

Linux (Ubuntu)

  1. Atualize o sistema: 
    sudo apt update
  2. Instale as dependências: 
    sudo apt install -y libmozjs185-1.0 libicu-dev libcurl4-openssl-dev
  3. Adicione o repositório CouchDB: 
    echo "deb https://apache.bintray.com/couchdb-deb focal main" | sudo tee -a /etc/apt/sources.list
  4. Atualize e instale o CouchDB: 
    sudo apt update
sudo apt install -y couchdb
  5. Siga as instruções para configurar o CouchDB (Standalone, Cluster, etc.).

Windows

  1. Baixe o instalador do site oficial do CouchDB.
  2. Execute o instalador e siga as instruções na tela.
  3. Após a instalação, o CouchDB será iniciado automaticamente.

Instalação em Container (Docker)

  1. Certifique-se de ter o Docker instalado e em execução.
  2. Puxe a imagem do CouchDB: 
    docker pull couchdb
  3. Execute o container: 
    docker run -d -p 5984:5984 --name couchdb -e COUCHDB_USER=admin -e COUCHDB_PASSWORD=password couchdb
