Criação do Banco de Dados

Vídeo-apresentação Sobre

Criando o Banco de Dados via Interface Web

  1. Abra seu navegador e acesse a interface web do CouchDB. Normalmente está em http://127.0.0.1:5984/_utils.
  2. Clique em "Create Database" e insira um nome único para seu banco de dados, como meu_banco_inedito.
  3. Clique em "Create" para criar o banco de dados.

Criando o Banco de Dados via curl

Se preferir usar curl, execute o seguinte comando no terminal:

curl -X PUT http://127.0.0.1:5984/meu_banco_inedito

Interação CRUD com o Banco de Dados

1. Create (Criar um documento)

Para criar um documento no banco de dados, você pode usar o seguinte comando curl:

curl -X POST http://127.0.0.1:5984/meu_banco_inedito -H "Content-Type: application/json" -d '{"nome": "João", "idade": 30, "cidade": "São Paulo"}'

2. Read (Ler um documento)

Para ler um documento do banco de dados, você precisa saber o ID do documento. Use o seguinte comando curl:

curl -X GET http://127.0.0.1:5984/meu_banco_inedito/{document_id}

3. Update (Atualizar um documento)

Para atualizar um documento, você precisa saber o ID e a revisão (rev) do documento. Use o seguinte comando curl:

curl -X PUT http://127.0.0.1:5984/meu_banco_inedito/{document_id} -H "Content-Type: application/json" -d '{"_rev": "{document_rev}", "nome": "João", "idade": 31, "cidade": "Rio de Janeiro"}'

4. Delete (Deletar um documento)

Para deletar um documento, você precisa saber o ID e a revisão (rev) do documento. Use o seguinte comando curl:

curl -X DELETE http://127.0.0.1:5984/meu_banco_inedito/{document_id}?rev={document_rev}
